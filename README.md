## Enunciado 01 - AgilStore

### Tecnologias utilizadas

- **Node**: Ambiente de execução JavaScript no servidor/backend.

- **Inquirer**: Biblioteca para criar a interface de linha de comando (CLI) interativa.

### Como rodar

**Requisitos**

- Node.js

---

1. Clone o repositório

```bash
git clone https://github.com/CaputiDev/desafios-aceleradora-agil-2026/
```

2. Acesse o diretório do exercício

```bash
cd exercicio-01-agilstore
```

3. Instale as dependências

```bash
npm install
```

4. Execute o programa

```bash
npm run dev
```
